# Zatoga
A PWA to easily get distracted with by:
* Playing Games such as: 
  * Doodle Jump
  * Chrome Dino
  * 2048 Game
  * Super Mario
* Playing an entire collection of more advanced online games which have been proxied


### Hosting:
